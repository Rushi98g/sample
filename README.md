# 🎓 AI Adoption & Student Performance Dashboard

A Power BI dashboard project that analyzes the impact of Generative AI usage on student academic performance, skill retention, anxiety levels, and burnout risk.



---

## 📌 Project Overview

This project explores how students use AI tools and how AI adoption influences academic outcomes. The dashboard provides insights into:

- Student AI usage patterns
- GPA improvements before and after AI adoption
- Skill retention scores
- Anxiety levels during exams
- Burnout risk distribution
- AI dependency trends
- Primary AI use cases across different student groups

---

## 📊 Dataset Information

The dataset contains student-level information including:

| Column | Description |
|----------|-------------|
| Student_ID | Unique student identifier |
| Major_Category | Student's academic discipline |
| Year_of_Study | Academic year |
| Pre_Semester_GPA | GPA before AI adoption |
| Post_Semester_GPA | GPA after AI adoption |
| Weekly_GenAI_Hours | Hours spent using AI weekly |
| Primary_Use_Case | Main AI usage purpose |
| Prompt_Engineering_Skill | Prompt engineering proficiency |
| Tool_Diversity | Number of AI tools used |
| Paid_Subscription | AI subscription status |
| Traditional_Study_Hours | Weekly study hours |
| Perceived_AI_Dependency | Self-reported dependency score |
| Institutional_Policy | College AI policy |
| Anxiety_Level_During_Exams | Anxiety score |
| Skill_Retention_Score | Knowledge retention score |
| Burnout_Risk_Level | Burnout category |

## 🔍 Key Insights

### Academic Performance
- Average GPA improved from **3.15** to **3.35** after AI adoption.
- Students with moderate AI usage generally achieved better academic outcomes.

### Skill Retention
- Advanced prompt engineering users showed higher retention scores.
- Effective AI usage can support learning rather than replace it.

### Anxiety & Burnout
- Average Anxiety Score: **4.27**
- Most students fall within the **Medium Burnout Risk** category.

### AI Usage Patterns
- STEM students recorded the highest average AI usage.
- Humanities students demonstrated the lowest AI usage levels.
